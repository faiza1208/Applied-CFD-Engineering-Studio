# Applied CFD Engineering Studio

Sprint 1 static website scaffold for a professional CFD engineering learning platform.

## Folder Structure

```text
.
├── index.html
├── modules.html
├── labs.html
├── resources.html
├── instructor.html
├── about.html
└── assets
    ├── css
    │   └── styles.css
    └── js
        ├── components.js
        └── main.js
```

## Local Preview

Because this is a static GitHub Pages-compatible site, you can preview it by opening `index.html` in a browser.

For a local server preview:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
